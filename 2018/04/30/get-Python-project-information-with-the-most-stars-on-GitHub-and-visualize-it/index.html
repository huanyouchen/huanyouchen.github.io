获取GitHub上星星最多的Python项目信息，并做可视化处理

《Python编程：入门到实践》中实践篇第二部分数据可视化的例子，目标是通过GitHub的API调用，获取GitHub上关于Python的星星数量最多的项目，并对其做可视化处理。

API地址：https://api.github.com/search/repositories?q=language:python&sort=stars

指定查询( q= )使用 language:python ,我们指出只想获取主要语言为Python 的仓库的信息。最后一部分( &sort=stars )指定将项目按其获得的星级进行排序。也可将语音改为JAVA，C等。
